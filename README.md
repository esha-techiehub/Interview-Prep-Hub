# Interview-Prep-Hub

Interview Prep Hub: curated questions and answers for DevOps (AWS, CI/CD, Kubernetes), Cyber Security (OWASP, SOC, IAM), and Data Analytics (SQL, Python, BI). Next.js app for students and working professionals.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

### Docker

```bash
docker compose up
```
